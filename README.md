# 薪資計算機 & 錢錢管家

個人財務管理工具組，包含薪資計算與日常記帳兩大功能，專為打工族、家教等兼職工作者設計。

## 功能簡介

### 薪資計算機 v2 (`薪資計算機v2.html`)

- 支援**時薪 / 堂薪 / 日薪**三種計費方式
- 管理多份工作，分別追蹤收入
- 月度儀表板：總收入、工作天數、收入分佈圖表
- 可設定每月目標薪資，追蹤進度
- 自動/手動同步薪資紀錄到錢錢管家

### 錢錢管家 v3 (`錢錢管家v3.html`)

- 記錄日常收入與支出
- 自訂支出類別（圖示 + 顏色）
- 多帳戶管理（現金 / 銀行 / 信用卡）
- 每月預算上限與進度追蹤
- 儲蓄目標管理
- 固定支出提醒（房租、訂閱等）
- 月度圖表分析（支出分佈、每日趨勢、上月比較）

### 共通功能

- 深色 / 淺色模式
- 行動裝置最佳化（支援加入主畫面）
- 6 種匯出格式：JSON、CSV、Markdown、Excel、Word、PDF
- 資料備份與還原（JSON 匯入/匯出）

## 快速開始

1. 下載或 clone 此專案
2. 用瀏覽器直接開啟 HTML 檔案即可使用
3. **不需要**安裝任何東西，不需要伺服器

> **提示：** 若要使用薪資同步功能，請從同一個資料夾開啟兩個 HTML 檔案。

## 同步機制

```
薪資計算機 v2 ──→ 錢錢管家 v3（單向同步）
```

- 薪資計算機新增紀錄後，可自動或手動同步到錢錢管家作為收入
- 同步的紀錄在錢錢管家中會標示 `[薪資]` 前綴
- 支援即時同步：兩個分頁同時開啟時，資料會透過 `storage` 事件即時更新

## 檔案結構

```
├── 薪資計算機v2.html    # 薪資計算機（主要版本）
├── 錢錢管家v3.html      # 記帳程式（主要版本）
├── 薪資計算機.html       # v1 舊版（保留）
├── 錢錢管家v2.html       # v2 舊版（保留）
├── 使用說明書.md         # 完整使用說明
├── 使用說明書.docx       # Word 版說明書
├── 使用說明書.pdf        # PDF 版說明書
├── build_docs.py         # 說明書產生腳本（docx）
└── build_pdf.py          # 說明書產生腳本（pdf）
```

## 技術細節

- 純前端，單一 HTML 檔案，零依賴安裝
- 資料儲存於瀏覽器 `localStorage`
- 外部程式庫（透過 CDN）：SheetJS、jsPDF、docx

## 注意事項

- 清除瀏覽器資料會導致紀錄遺失，請定期備份
- 目前不支援雲端同步，跨裝置需手動匯出/匯入 JSON
- 不同瀏覽器的資料互相獨立
