# 薪資計算機 & 錢錢管家 — 使用說明書

> 版本：v2.0 / v3.0　更新日期：2026-04-13

---

## 一、概述

本專案包含兩個互相串接的行動網頁應用程式：

| 應用程式 | 檔案 | 用途 |
|----------|------|------|
| **薪資計算機 v2** | `薪資計算機v2.html` | 打工、家教薪資紀錄與計算 |
| **錢錢管家 v3** | `錢錢管家v3.html` | 日常收支記帳與預算管理 |

兩者皆為單一 HTML 檔案，無需安裝，使用瀏覽器直接開啟即可。資料儲存在瀏覽器的 localStorage 中。

### 舊版檔案（保留用）

| 檔案 | 說明 |
|------|------|
| `薪資計算機.html` | v1 原版（SVG 風格、底部導航） |
| `錢錢管家v2.html` | v2 原版（僅 CSV + JSON 匯出） |

---

## 二、快速開始

### 1. 開啟方式

直接在瀏覽器中開啟 HTML 檔案即可使用。

> **重要：** 若要使用兩個程式之間的「同步」功能，必須從**相同來源**開啟兩個檔案（例如同一個資料夾用 `file:///` 協議開啟，或透過同一個本地伺服器）。

### 2. 手機使用

兩個程式皆為行動裝置最佳化設計（480px 寬度），支援：
- iOS Safari「加入主畫面」（PWA-like）
- 深色 / 淺色模式自動同步
- 安全區域適配（瀏海手機）

## 三、薪資計算機 v2 功能說明

### 導航列

| 頁籤 | 圖示 | 功能 |
|------|------|------|
| 總覽 | 📊 | 月度收入摘要、圖表、目標進度 |
| 記帳 | ✏️ | 新增薪資紀錄 |
| 明細 | 📋 | 瀏覽、搜尋、刪除紀錄 |
| 工作 | 💼 | 管理工作類型 |
| 設定 | ⚙️ | 主題、目標薪資、同步、匯出 |

### 3.1 工作管理（💼 工作）

使用前需先建立工作類型。

**新增工作：**
1. 前往「工作」頁籤
2. 在下方「新增工作」區域填入：
   - **工作名稱**：例如「王同學家教」、「咖啡店打工」
   - **計費方式**：
     - **時薪**：按工作時數計算（扣除休息時間）
     - **堂薪**：按上課堂數計算
     - **日薪**：按天計算固定金額
   - **費率**：對應的金額（時薪金額 / 每堂金額 / 每日金額）
   - **顏色**：用於圖表和識別
   - **每堂時長**（僅堂薪）：每堂課的時間長度
3. 點擊「新增工作」

**編輯/停用/刪除工作：**
- 點擊工作旁的「編輯」按鈕
- 可修改所有欄位
- 「停用」後該工作不會出現在記帳頁面的選擇中
- 「刪除」會永久移除（已存在的紀錄不受影響）

### 3.2 記帳（✏️ 記帳）

**新增紀錄：**
1. 選擇工作（藥丸按鈕）
2. 選擇日期
3. 根據計費方式填入資訊：
   - **時薪**：開始時間、結束時間、休息時間（可選 0/30/60/90 分鐘或自訂）
   - **堂薪**：堂數
   - **日薪**：僅需日期
4. 預計薪資會即時顯示
5. 可加入備註（選填）
6. 點擊「✅ 新增」

**薪資計算邏輯：**
- 時薪：(工作時長 - 休息時間) ÷ 60 × 時薪費率
- 堂薪：堂數 × 堂薪費率
- 日薪：日薪費率（固定）

### 3.3 總覽（📊 總覽）

- **月度摘要卡片**：總收入、工作天數、平均日薪
- **目標進度**：若已設定目標薪資，顯示進度條
- **收入分佈**：環形圖，按工作分類顯示收入比例
- **每日收入**：長條圖，顯示每天的收入分佈
- **工作摘要**：各工作的累計時數/堂數與總薪資

使用月份切換按鈕（◀ ▶）切換檢視不同月份。

### 3.4 明細（📋 明細）

- 顯示當月所有紀錄
- 搜尋功能：可搜尋工作名稱或備註
- 每筆紀錄顯示：工作名稱、日期、時長/堂數、薪資
- 已同步紀錄會顯示「已同步」標籤
- 點擊 ✕ 可刪除紀錄
  - 若紀錄已同步，會提醒使用者至錢錢管家手動移除

### 3.5 設定（⚙️ 設定）

- **主題模式**：深色 / 淺色切換（與錢錢管家同步）
- **每月目標薪資**：設定後會在總覽顯示進度
- **同步至錢錢管家**：詳見第五章

## 四、錢錢管家 v3 功能說明

### 導航列

| 頁籤 | 圖示 | 功能 |
|------|------|------|
| 總覽 | 📊 | 月度收支摘要、圖表 |
| 記帳 | ✏️ | 新增收入/支出 |
| 明細 | 📋 | 瀏覽、搜尋、刪除紀錄 |
| 目標 | 🎯 | 儲蓄目標、固定支出 |
| 設定 | ⚙️ | 主題、預算、類別、匯出 |

### 4.1 記帳（✏️ 記帳）

1. 選擇類型：💸 支出 / 💰 收入
2. 輸入金額
3. 選擇帳戶：現金 / 銀行 / 信用卡
4. 選擇類別（僅支出）：飲食、交通、購物、娛樂、帳單、醫療、教育、其他
5. 選擇日期
6. 備註（選填）
7. 點擊「✅ 新增」

### 4.2 總覽（📊 總覽）

- **收支摘要**：支出（紅）、收入（綠）、結餘（藍）
- **上月比較**：與上月支出的差異
- **預算進度**：若已設定預算，顯示使用進度
- **支出分佈**：環形圖，按類別顯示（含上月比較箭頭）
- **每日支出**：長條圖
- **帳戶分佈**：各帳戶的收支情況
- **近期固定支出提醒**：距今 3 日內的固定支出

### 4.3 明細（📋 明細）

- 月度紀錄列表，支援搜尋
- 來自薪資計算機同步的紀錄會顯示「薪資計算機」標籤
- 點擊 ✕ 可刪除

### 4.4 目標（🎯 目標）

**儲蓄目標：**
- 建立目標（名稱 + 目標金額 + 已存金額）
- 顯示進度條
- 可隨時「+存入」或「-取出」
- 達標時顯示 🎉

**固定支出：**
- 建立固定支出（名稱、金額、每月幾號）
- 支援「快速記帳」一鍵記錄
- 到期前 3 天自動提醒

### 4.5 設定（⚙️ 設定）

- **主題模式**：深色 / 淺色
- **每月預算上限**
- **自訂類別**：新增自訂支出類別（選擇圖示 + 顏色）
- **帳戶管理**：新增自訂帳戶
- **資料管理**：匯出、匯入、清除（詳見第六章）

## 五、同步機制

### 同步方向

```
薪資計算機 ──→ 錢錢管家（單向）
```

薪資計算機的紀錄可以同步到錢錢管家作為「收入」紀錄。反向不支援。

### 同步模式

在薪資計算機的「設定 → 同步至錢錢管家」中選擇：

| 模式 | 說明 |
|------|------|
| **自動** | 每次新增紀錄時自動同步 |
| **手動** | 手動點擊「同步」按鈕才同步 |

### 同步後的紀錄

- 在錢錢管家的明細中，同步來的紀錄備註會以 `[薪資]` 開頭
- 明細頁面會顯示「薪資計算機」標籤
- 帳戶預設為「銀行」
- 類別為「收入」

### 注意事項

- 同步為**單向**，修改或刪除已同步的紀錄時，需至對應的程式手動處理
- 刪除已同步紀錄時，薪資計算機會彈出提醒
- 同步統計數字顯示在設定頁面（已同步 / 未同步筆數）

## 六、匯出功能

兩個程式皆支援 **6 種匯出格式**：

| 格式 | 副檔名 | 說明 |
|------|--------|------|
| JSON | `.json` | 完整結構化資料，可用於備份還原 |
| CSV | `.csv` | Excel / Google Sheets 可開啟 |
| Markdown | `.md` | 純文字表格，適合筆記軟體 |
| Excel | `.xlsx` | 含紀錄 + 摘要雙工作表 |
| Word | `.docx` | 排版好的表格文件 |
| PDF | `.pdf` | 橫向排版，適合列印或分享 |

### 薪資計算機的額外匯出功能

- **工作篩選**：可選擇只匯出特定工作的紀錄（例如只匯出某位學生的家教紀錄，方便和家長報帳）
- **匯出錢錢管家 JSON 格式**：將薪資資料轉換為錢錢管家可匯入的格式，用於異地備份還原

### 匯出範圍

- **當月紀錄**：僅匯出目前檢視月份的紀錄
- **全部紀錄**：匯出所有歷史紀錄

## 七、資料備份與還原

### 匯出備份

1. 前往「設定 → 資料管理」
2. 選擇 JSON 格式匯出
3. 儲存下載的 `.json` 檔案

### 匯入還原

1. 前往「設定 → 資料管理」
2. 點擊「📥 匯入備份」
3. 選擇先前匯出的 `.json` 檔案
4. 資料會完全覆蓋現有資料

### 清除資料

- 點擊「🗑️ 清除資料」會刪除所有紀錄
- 此操作**不可還原**，請先備份

兩個程式皆採用模組化架構，便於未來擴充：

- **資料層與 UI 層分離**：資料操作函數獨立於 `render()` 渲染函數
- **統一主題系統**：`T()` 函數回傳完整色彩配置

## 十、常見問題

**Q: 兩個程式的資料不互通怎麼辦？**
A: 請確認兩個 HTML 檔案是從相同來源（同一資料夾）開啟的。不同來源的 localStorage 是隔離的。

**Q: 手機上資料會消失嗎？**
A: localStorage 在清除瀏覽器資料時會被刪除。建議定期使用 JSON 匯出功能備份。

**Q: 可以多裝置同步嗎？**
A: 目前不支援雲端同步。可透過匯出 JSON → 傳送到另一裝置 → 匯入的方式手動同步。

**Q: 匯出的 PDF 中文顯示為方塊？**
A: PDF 使用英文標題以避免字體問題。表格內的數字和日期可正常顯示。
